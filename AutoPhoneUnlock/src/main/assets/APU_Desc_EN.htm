(1)        (Except Android4.2)To bypass the key guard at the screen turned on if it is connected to a trusted device (wireless LAN or Bluetooth) or Bluetooth Low Energy tag exists near. (key guard will appear if it is not already connected)

However, it is necessary release the key guard of only manually first once if you have connected a reliable device when the screen turns off.

(2)        On and off the screen in the proximity sensor

(4)        To turn off the wireless LAN, if does not connect to the access point within a specified time.

(5)        To turn off the Bluetooth, if does not connect to the device within a specified time.

2.Configuration

Operation is done by tapping the icon that appears on the screen. Screen is divided into basic mode and selection mode, you can switch to select from basic in that you press and hold an item in the tap or screen the entire selection icon. To switch to basic mode by pressing the return key.

 

2.1Trusted Device Tab

Add and remove a trusted device. Operation is done by tapping the icon that appears on the screen. Screen is divided into basic mode and selection mode, you can switch to select from basic in that you press and hold an item in the tap or screen the entire selection icon.

To switch to basic mode by pressing the return key.

(1)        Options

①     To immediately lock when all trusted device was disconnected

All of trusted device is turned off and lock the screen screen when you are disconnected.

②     Time to a lock state when the terminal is disconnected from all the trusted devices

It is not locked when all trusted devices have been disconnected are connected any trusted devices within the waiting time.

③     Use screen lock by application password

You use a password on the screen of the locking and unlocking. Please use in case a problem occurs on the screen of the unlock.

 

 

(2)        Trusted device

①     Basic mode

  Show Add device dialog

  Select all device

 Enable if switch is ON.

Edit(see 2.1.2 Editing of trusted devices)

Send alarm to Bluetooth low energy tag

②     Selection mode

 Delete device

 Enable device(Eligible for the trusted device)

 Disable device(Exclude the trusted device)

 Select all device

 Unselect all device

 

2.1.1Add trusted device

Add a trusted device. You can switch between the selection and non-selection when you tap the device you want to add.

You can add by pressing the Add button to select the device. Device that has been added will be immediately reflected in the system.

 

 

2.1.2 Editing of trusted devices

Will be displayed that the edit screen to tap a line of device.

ü  Item name

Freely given the name of the registered device (duplication and blank is also possible)

ü  Type

Device type

ü  Device name

SSID name of WiFi access point or Device name of bluetooth

ü  Device add

SSID MAC address of WiFi access point or Device MAC address of bluetooth

ü  Device connection

       "Advertising reception" is determined to be when receiving the advertising signal tag is connected to the trusted device. "Device connected" determined to be connected to a trusted device only when the terminal and the tags are connected.

       "Device connection" is possible to control the 1 In a terminal in 1 tag. "Advertising reception" is possible to control a plurality of terminals in one tag. Please use the "advertising reception" normal. "Device connection" Please use when the tag is in the tag is turned off when not connected to the terminal.

2.2 Screen Tab

Set the screen control by the proximity sensor.

2.2.1 Proximity sensor

(1)        Temporarily disable when it receives five or more events in 10 seconds

Temporarily you want to disable the lighting of the screen when the lighting and off of the screen has been frequently repeated.

To enable it to tap the message in the notification area.

(2)        Turn on the screen when sensor is not sense the proximity when screen is off

This is a function to turn on the screen when it is out of the pocket or bag.

(3)        Lock the screen when sensor is sense the proximity beyond the specified time

This is a function to turn off the screen when it is put in a pocket or bag.

ü  Ignore proximity sensor when device orientation is landscape

This is a function to prevent the lock screen when the camera is shooting.

ü  Proximity sensor time

The screen time to turn off.
